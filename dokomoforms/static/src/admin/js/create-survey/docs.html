DokomoForms Documentation

1. Survey Authorship

Users with admininstrator privileges can publish surveys directly from the administration panel to enumerators' mobile devices.


1.1 Survey Title
The title of the survey must be unique and cannot have been previously used as the title of another survey.


1.2 Survey Type
Specify who has access to this survey.
Public - anyone can access this survey.
Enumerator only - only assigned enumerators can access this survey.


1.3 Default Language
Specify the language of the survey.
Survey title, question title, question hints, answers for a survey must all be in the same language.
If no language is specified, the default language will be English.


1.4 Custom Numbering (optional) (recommended)
Add numbers to questions. If using custom numbering, all questions in the survey must have a number.
Both numbers and letters can be used.
Must be unique, no question can have the same number in a single survey.
If custom numbering is not used, the questions will not be numbered.


2. Questions
Each survey (including sub-surveys) must have at least one question.

2.1 Question Title
The text of the question that will be asked to the survey receipient.

2.2 Type Constraint / Answer Type
Specifies the type of answer that must be given. Answers that are not compatible with the type constraint will not be accepted. Options are:

  • Text
  • Integer
  • Decimal
  • Date
  • Time


2.2.1 Multiple Choice
If the survey author chooses 'multiple choice' as a question's type constraint, they must provide a list of choices for the survey recepient to choose from. At least two choices must be given.
By choosing the "allow other" option, the survey author can allow the survey recepient to choose "other" as an answer if none of the given answers are applicable.


2.3 Question Hint (optional)
The question hint. Can be used to indicate expected answer type, if multiple responses can be given, etc.

3. Sub-Surveys
A sub-survey is a list of questions that branches off its parent survey.

Usage: Buckets: The survey author can indicate a specific answer (in the case of a multiple choice question) or range of answers (in the case of integer, decimal, time, or date question) that if chosen, can lead to the sub-survey. This specified answer is referred to as a bucket. Skip to section 4 for more information on buckets.
For example, if the question is "do you have children?" the survey author can indicate that answering "yes" is a bucket to a sub-survey of questions that asks about the survey recepient's children. After the sub-survey is completed, the survey will automatically skip back to the following question in the parent survey.
If the receipient answers "no", the sub-survey will not be given and the next question in the parent survey will be shown.

Repeating: The survey author can indicate that a sub-survey is repeatable. The number of times a sub-survey repeats will be based on the answer to the parent question.
For parent questions with an integer type constraint, the sub-survey will repeat the number of times of the given answer.
For parent questions that allow multiple responses, the sub-survey will repeat as many times as there are given answers.
Sub-surveys can not be both repeatable and have buckets.
A parent question can have multiple sub-surveys.
Sub-surveys can not have the same bucket lead to two separate sub-surveys.
A sub-survey can, however, have multiple buckets that lead to it.
Sub-surveys can branch off any survey, including other sub-surveys.
If a sub-survey does not have buckets, it must be repeatable.

4. Buckets
The survey author can indicate a specific answer (in the case of a multiple choice question) or range of answers (in the case of integer, decimal, time, or date question) that if chosen, can lead to the sub-survey. This specified answer is referred to as a bucket.

Bucket Types

Multiple Choice:

One or more of the choices listed in a multiple choice question. If one of the specified buckets is selected, the receipient will be lead to the sub-survey.

Range:

For an integer or decimal question. If the receipients gives an answer that falls in the range, they will be lead to the sub-survey. A minimum and a maximum in the range must be specified. Minimum defaults to 0 and maximum at 100.

Multiple buckets can lead to the same sub-survey, but one bucket can not lead to multiple sub-surveys.
